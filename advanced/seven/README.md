#### Server Information

#### Color Scheme
```javascript
// Modify the Tailwind config in index.html

    'np': {
        'blue': '#1e3a8a',      // Primary blue

        'gray': '#334155',       // Body text
        'light-gray': '#64748b'  // Muted text
    }
}
```
